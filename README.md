# Ola_Dashboard

Ola Ride Analytics Dashboard
1. Project Title / Headline

Ola Ride Analytics Dashboard: Smart Mobility Insights
An interactive and dynamic Power BI dashboard designed to analyze Ola ride data—focusing on bookings, cancellations, revenue trends, customer behavior, and operational performance.

2. Short Description / Purpose

The Ola Ride Analytics Dashboard is a data-driven Power BI report that helps analyze ride bookings, cancellations, revenue, and customer trends. It is designed to provide actionable insights for improving ride efficiency, customer satisfaction, and business decision-making.

3. Tech Stack

The dashboard was built using the following tools and technologies:

• 📊 Power BI Desktop – Used for creating interactive dashboards and visualizations
• 📂 Power Query – Data cleaning and transformation
• 🧠 DAX (Data Analysis Expressions) – Used for calculated measures and KPIs
• 🔗 Data Modeling – Relationships between booking, customer, and ride data
• 📁 File Format – .pbix (Power BI file) and .png (dashboard preview)

4. Data Source

The dataset used in this project is based on Ola ride booking data, which includes:

Booking ID
Ride Status (Completed, Cancelled by Customer/Driver)
Customer details
Vehicle type
Booking value (revenue)
Ride distance and duration

This structured dataset allows analysis of ride patterns, cancellations, and revenue generation.

5. Features / Highlights
• Business Problem

Ride-hailing platforms like Ola generate massive amounts of data daily, but without proper visualization, it becomes difficult to analyze:

Cancellation patterns
Revenue performance
Customer behavior
Operational efficiency


• Goal of the Dashboard

To build an interactive dashboard that:

Tracks total bookings and revenue
Analyzes cancellations by customers and drivers
Identifies peak booking trends
Provides insights into ride performance and efficiency
